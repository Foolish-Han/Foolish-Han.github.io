选数异或-2081

给定一个长度为 nnn 的数列 A1,A2,...,AnA_1,A_2,...,A_nA1​,A2​,...,An​ 和一个非负整数 xxx, 给定 mmm 次查询, 每次询问能否从某个区间 [l,r][l, r][l,r] 中选择两个数使得他们的异或等于 xxx 。

有一说一，只要想明白了，这个题其实并不难。但可惜的是，我在做题的时候囿于定论，看到题目下方“线段树”的标签，便苦思冥想试图用线段树去解决问题，最终也没有憋出个所以然来。看了题解之后，豁然开朗，才感觉到自己的愚笨。

最基本的，要判断区间是否满足条件，实际上等同于在区间内存在一个数 yyy，且 y⊕xy \oplus xy⊕x 也在区间内。那么最基本的，我们可以用一个数组 record[n]record[n]record[n] 去记录每个数左端最近的匹配数。即 record[i]=j(j<i)record[i] = j (j < i)record[i]=j(j<i) 等价于 Ai⊕Aj=xA_i \oplus A_j = xAi​⊕Aj​=x 且不存在 j<k<ij < k < ij<k<i 使得 Ai⊕Ak=xA_i \oplus A_k = xAi​⊕Ak​=x。这样，我们在判断区间 [l,r][l, r][l,r] 是否符合条件时只需遍历 record[i],l≤i≤rrecord[i], l \le i \le rrecord[i],l≤i≤r 判断其是否在 [l,r][l, r][l,r] 内即可。

然而这样每次 O(n)O(n)O(n) 的查询代价太大了，是否可以进一步优化呢？实际上，我们并不关心某一个数的匹配数，我们关心的是区间内是否存在匹配数。那么我们就可以定义 record[i]=jrecord[i]=jrecord[i]=j 表示区间 [j,i][j, i][j,i] 符合要求且不存在 [k,i],j<k<i[k,i],j < k <i[k,i],j<k<i 符合要求。则相应的有转移方程 record[i+1]=max⁡{record[i],last[x⊕Ai+1]}record[i+1] = \max \{record[i], last[x \oplus A_{i+1}] \}record[i+1]=max{record[i],last[x⊕Ai+1​]}。

算法问题一定要放开想，要不就会被一道简简单单的动态规划题难住了…

爬树的甲壳虫

有一只甲壳虫想要爬上一颗高度为 nnn 的树，它一开始位于树根，高度为 000，

当它尝试从高度 i−1i−1i−1 爬到高度为 iii 的位置时有 PiP_iPi​ 的概率会掉回树根，求它从树根爬到树顶时，经过的时间的期望值是多少。

输入第一行包含一个整数 nnn表示树的高度。

接下来 nnn 行每行包含两个整数 nnn，用一个空格分隔，表示 Pi=xiyiP_i = \frac{x_i}{y_i}Pi​=yi​xi​​。

输出一行包含一个整数表示答案, 答案是一个有理数, 请输出答案对质 数 998244353998244353998244353 取模的结果。

其中有理数 ab\frac{a}{b}ba​ 对质数 PPP 取模的结果是整数 ccc 满足 0≤c≤P0 \le c \le P0≤c≤P 且 c⋅b≡a(modP)c \cdot b \equiv a (mod P)c⋅b≡a(modP)。

由于是第一次做这种期望 DP 题（也是第一次见），自然是没有什么思路。这个题的难点有两个，一是取模运算中对除法的处理，二是转移方程的寻找。

逆元

关于模运算，我们很清楚的是关于加减乘法的运算性质。需要注意的是，模运算是不允许出现负数的，因此需要加上 ppp 使结果变为非负数。比如 (3−7)mod  5=(3mod  5−7mod  5+5)mod  5=(3−2+5)mod  5=6mod  5=1(3 - 7) \mod 5 = (3 \mod 5 - 7 \mod 5 + 5) \mod 5 = (3 - 2 + 5) \mod 5 = 6 \mod 5 = 1(3−7)mod5=(3mod5−7mod5+5)mod5=(3−2+5)mod5=6mod5=1

(a+b)mod  p=[(amod  p)+(bmod  p)]mod  p(a+b) \mod p = [(a \mod p) + (b \mod p)] \mod p(a+b)modp=[(amodp)+(bmodp)]modp

(a−b)mod  p=[(amod  p)−(bmod  p)+p]mod  p(a-b) \mod p = [(a \mod p) - (b \mod p) + p] \mod p(a−b)modp=[(amodp)−(bmodp)+p]modp

(a×b)mod  p=[(amod  p)×(bmod  p)]mod  p(a \times b) \mod p = [(a \mod p) \times (b \mod p)] \mod p(a×b)modp=[(amodp)×(bmodp)]modp

(a×b)×c≡a×(b×c)(modp)(a \times b) \times c \equiv a \times (b \times c) \pmod{p}(a×b)×c≡a×(b×c)(modp)

a×b≡b×a(modp)a \times b \equiv b \times a \pmod{p}a×b≡b×a(modp)

那么有没有关于除法的运算性质呢？

在整数运算中，除法是乘法的逆运算，也即 ab=c\frac{a}{b}=cba​=c 等价于 a=b×ca = b \times ca=b×c。但是，在模 ppp 意义下，我们不能直接做除法，而是需要转换成乘法。 即，在模 ppp 下，如果我们要计算：

acmod  p\frac{a}{c} \mod pca​modp

等价于我们要找到一个数 xxx 使得：

a÷c≡x(modp)a \div c \equiv x \pmod{p}a÷c≡x(modp)

这实际上等价于：

a≡x×c(modp)a \equiv x \times c \pmod{p}a≡x×c(modp)

在模 ppp 下，除法就是找一个数，使得乘回去得到原数，这正是乘法逆元的定义。

数 ccc 在模 ppp 下的逆元 c−1c^{-1}c−1 是满足：

c×c−1≡1(modp)c \times c^{-1} \equiv 1 \pmod{p}c×c−1≡1(modp)

的数。

如果我们将除法转换成乘法逆元运算：

x=a×c−1mod  px = a \times c^{-1} \mod px=a×c−1modp

那么验证一下：

a×c−1×c≡a×1≡a(modp)a \times c^{-1} \times c \equiv a \times 1 \equiv a \pmod{p}a×c−1×c≡a×1≡a(modp)

这和原来的等式 a≡x×c(modp)a \equiv x \times c \pmod{p}a≡x×c(modp) 是一致的，因此：

acmod  p=a×c−1mod  p\frac{a}{c} \mod p = a \times c^{-1} \mod pca​modp=a×c−1modp

那么逆元该怎么求呢？我们都知道最基本的寻找两个数的最大公因数的方法——欧几里得方法，即 gcd(a,b)=gcd(b,amod  b)gcd(a,b)=gcd(b,a \mod b)gcd(a,b)=gcd(b,amodb) 直到 b=0b = 0b=0 时，aaa 即为最大公因数。

扩展欧几里得算法不仅求 gcd⁡(a,b)\gcd(a, b)gcd(a,b)，还可以找到整数 xxx 和 yyy，使得：

ax+by=gcd⁡(a,b)a x + b y = \gcd(a, b)ax+by=gcd(a,b)

假设：

gcd⁡(b,amod  b)=bx′+(amod  b)y′\gcd(b, a \mod b) = b x' + (a \mod b) y'gcd(b,amodb)=bx′+(amodb)y′

根据模运算的定义有：

amod  b=a−⌊a/b⌋×ba \mod b = a - \lfloor a / b \rfloor \times bamodb=a−⌊a/b⌋×b

代入整理得：

gcd⁡(a,b)=ay′+b(x′−⌊a/b⌋×y′)\gcd(a, b) = a y' + b (x' - \lfloor a / b \rfloor \times y')gcd(a,b)=ay′+b(x′−⌊a/b⌋×y′)

由此可得：

x=y′x = y'x=y′

y′=x′−⌊a/b⌋×y′y' = x' - \lfloor a / b \rfloor \times y'y′=x′−⌊a/b⌋×y′

我们如何确保存在对应的 xxx 和 yyy 呢？因为递归的出口 b=0b = 0b=0 时，必然存在 x=1x = 1x=1 和 y=0y = 0y=0 这组解。

于是我们有如下扩展欧几里得算法：

在模运算下，如果 aaa 和 ppp 互质，则有 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1，那么根据扩展欧几里得算法：

ax+py=gcd⁡(a,p)=1a x + p y = \gcd(a, p) = 1ax+py=gcd(a,p)=1

取模 ppp：

ax≡1(modp)a x \equiv 1 \pmod{p}ax≡1(modp)

因此，xxx 就是 a−1a^{-1}a−1。

不过在实际中，我们更常用的是费马小定理计算逆元。

如果 ppp 是素数（注意，这里的条件相对于扩展欧几里得算法里的两数互质的要求较为苛刻），则根据费马小定理：

ap−1≡1(modp)a^{p-1} \equiv 1 \pmod{p}ap−1≡1(modp)

两边同时除以 aaa，可得：

ap−2≡a−1(modp)a^{p-2} \equiv a^{-1} \pmod{p}ap−2≡a−1(modp)

这意味着：

a−1=ap−2mod  pa^{-1} = a^{p-2} \mod pa−1=ap−2modp

这样，我们可以用快速幂算法计算 ap−2mod  pa^{p-2} \mod pap−2modp 来得到逆元。
